# Weather Diary - Дневник погоды

**Автор:** Левыкин Александр

## 📝 Описание

Weather Diary - это графическое приложение для ведения дневника погоды. Позволяет добавлять записи о погоде, просматривать их, фильтровать по дате и температуре, а также редактировать и удалять записи. Все данные сохраняются в JSON-файл.

## ✨ Основные возможности

- ✅ Добавление записей о погоде (дата, температура, описание, осадки)
- 🔍 Фильтрация записей по дате
- 🌡️ Фильтрация записей по температуре (выше +10°C / ниже или равно 0°C)
- 📊 Просмотр статистики (количество записей)
- ✏️ Редактирование существующих записей
- 🗑️ Удаление записей
- 💾 Автоматическое сохранение в JSON-файл
- 📋 Валидация ввода данных
